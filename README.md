# Wheelbase server

![GitHub Actions Workflow Status](https://img.shields.io/github/actions/workflow/status/julien-rg/wheelbase-server/.github%2Fworkflows%2Fci.yml?branch=main)

This repository contains the source code for the Wheelbase server, which exposes a REST API for the Wheelbase application.
The server is built using Spring Boot and provides the backend functionality required by the Wheelbase app.
