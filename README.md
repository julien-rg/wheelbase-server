#### This repository contains the source code for the Wheelbase server, which exposes a REST API for the Wheelbase application.

------------

##### The server is built using Spring Boot and provides the backend functionality required by the Wheelbase app, including:
-  User authentication and authorization
- JWT-based security
- Data management via RESTful endpoints
- Business logic for the Wheelbase application
- Unit tests for core services
- CI/CD workflows for automated testing and code quality checks
